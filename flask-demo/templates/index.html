Cedar Flask Demo

This is a simple demo that showcases how you can integrate Cedar into your Python applications. This mimics a simple photo sharing application, which has a number of different access requirements. There is public access, user access, user admin access, and administrate access, all of which unlock certain parts of the site (for simplicity it is just the top level menus).

You can login using a number of sample users (check the users.dat file). These all have different levels of access. The level of access is defined by the user (or entity) and the cedar policy. You can view these files and update to experiment.

What is Cedar

Cedar is a language for defining permissions as policies, which describe who should have access to what. It is also a specification for evaluating those policies. Use Cedar policies to control what each user of your application is permitted to do and what resources they may access.

This is a simple demo of how you can use Cedar with Python. It uses an open source project from k9 Security, called cedar-py
